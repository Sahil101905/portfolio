# Sahil Khan — Engineering Case Studies

A cinematic engineering portfolio documenting how I move from technical constraints to tested systems across mechanical engineering, biomedical engineering, and software.

## [View the live portfolio →](https://sahil-khan-engineering-portfolio.skhan101905.chatgpt.site/)

## Featured work

- Tetra-chiral prosthetic lattice research
- Modular prosthetic hand with EMG intent prediction
- Finite-element analysis software platform
- Siemens NX-to-MATLAB CAD automation and optimization pipeline
- A high-level overview of my work with [REEF](https://reef.bio)

The project pages emphasize the problem, requirements, engineering decisions, verification, results, and lessons learned. Private and organization-sensitive work is represented through sanitized summaries and generalized architecture only.

## Portfolio stack

- TypeScript and React
- Vinext and Vite
- Tailwind CSS
- Cloudflare-compatible deployment through Sites

## Run locally

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
